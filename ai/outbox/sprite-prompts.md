# 코인던전 캐릭터 스프라이트 생성 프롬프트

## 사용법
1. ChatGPT (GPT-4 + DALL-E)에 아래 프롬프트를 **하나씩** 붙여넣기
2. 생성된 이미지를 다운로드
3. 필요시 배경 제거 (remove.bg 또는 Photoshop)
4. 96x128px로 리사이즈 (정확한 크기 중요!)
5. `public/assets/sprites/characters/` 폴더에 저장

---

## 공통 지시 (먼저 GPT에 한번 보내기)

```
너는 게임 스프라이트 아티스트야.
나는 도트풍 2D RPG 게임의 캐릭터 스프라이트시트를 만들어야 해.

규격:
- 최종 이미지 크기: 96x128 픽셀 (가로 96, 세로 128)
- 프레임 크기: 32x32 픽셀
- 배치: 3열 x 4행 = 총 12프레임
- 행 배치:
  - 1행 (맨 위): 아래쪽 보기 걷기 3프레임
  - 2행: 왼쪽 보기 걷기 3프레임
  - 3행: 오른쪽 보기 걷기 3프레임
  - 4행 (맨 아래): 위쪽 보기 걷기 3프레임
- 각 행의 3프레임: 서기 → 왼발 → 오른발 (걷기 사이클)
- 스타일: 16-bit 레트로 도트(pixel art), 굵은 외곽선
- 배경: 투명 또는 단색(나중에 제거 가능)
- 캐릭터 크기: 프레임 내 약 20x28px (위아래 여백 있게)

이 규격을 이해했으면 "준비 완료"라고 답해. 이후 캐릭터별로 하나씩 요청할게.
```

## 후처리 안내

### DALL-E 결과물이 정확하지 않을 경우

DALL-E는 정확한 픽셀 배치를 보장하지 못할 수 있음. 그럴 경우:

**방법 A: GPT에게 수정 요청**
```
이 이미지를 정확히 96x128 픽셀로, 3열 4행 그리드에 맞춰서 다시 그려줘.
각 프레임이 정확히 32x32 픽셀이어야 해.
```

**방법 B: 무료 도트 에디터로 직접 수정**
- [Piskel](https://www.piskelapp.com/) — 웹 브라우저에서 바로 사용
- [Pixilart](https://www.pixilart.com/) — 온라인 도트 에디터
- [LibreSprite](https://libresprite.github.io/) — Aseprite 무료 대안

**방법 C: AI + 수동 조합**
1. GPT로 캐릭터 정면 1장만 생성
2. Piskel에서 96x128 캔버스 만들기
3. AI 이미지를 참고해서 12프레임 직접 배치

### 배경 제거
- [remove.bg](https://www.remove.bg/) — 무료 배경 제거
- 또는 GPT에 "transparent background" 강조

### 리사이즈
- 정확히 96x128px인지 확인 (크면 리사이즈)
- 리사이즈 시 **Nearest Neighbor** 보간법 사용 (도트가 뭉개지지 않게)
